### For Mac OS
1. Install Homebrew
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
2. Ready to use git
```
brew install git
```
```
git config --global user.name marronnier-tech
git config --global user.email "Email Address"
```

3. Execute script
```
sh con-env.sh
```
